20/*!
21@brief Calculates Structure Radiation + Normalisation of states, using diagram
22method.
23@details
24Three functions: srTB(), srC(), norm().
25Structure radiation is sum of srTB()+srC(); norm() gives normalisation of
26states.
27
28 - Typically, one should use splines for the 'legs' (outer states) of diagrams.
29 However, code is written such that {w,v} (the valence states) always appear as
30the _first_ state in the Q integrals; thus those states are always directly
31integrated (other states may use the existing y^k integrals, which were
32calculated using the spline states) * unless using QkTable (see below).
33
34 - For using spline states as legs: This means you must typically ensure basis
35is large enough to make relevant valence states "physical"
36
37 - The user should check if the zeroth order <w|t|v> matches closely between
38valence/spline states. If not, basis/cavity is too small and SR+N probably
39meaningless
40
41 - Option to use QkTable to calculate SR+N. Otherwise, calculates Coulomb
42integrals on-the-fly. If QkTable is used, leads to ~10x speed-up, but uses large
43amount of memory. Also, using QkTable means spline states are used for "legs" of
44diagrams.
45
46
47 - Formulas are presented in (for example):
48 * W. R. Johnson, Z. W. Liu, and J. Sapirstein, At. Data Nucl. Data Tables 64,
49279 (1996). doi:10.1006/adnd.1996.0024
50
51*/

59 nminmax is a pair{min, max}: we only used core states with n>=min, and only
60 uses excited states with n<=nmax in the summations.
61 Qk_fname is filename for QkTable - if given it will read/write QkTable to this
62 file, and use QkTable to calculate SR. This ldeads to ~10x speedup in

112 srn(const DiracOperator::TensorOperator *const h, const DiracSpinor &w,
113 const DiracSpinor &v, double omega = 0.0,
114 const ExternalField::CorePolarisation *const dV = nullptr) const {
115 const auto [tb, dvtb] = srTB(h, w, v, omega, dV);
116 const auto [c, dvc] = srC(h, w, v, dV);
117 const auto [n, dvn] = norm(h, w, v, dV);
118 return {tb + c + n, dvtb + dvc + dvn};
