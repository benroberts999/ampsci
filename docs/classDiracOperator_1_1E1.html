Detailed Description

Electric dipole operator: -|e|r = -er.

\[<a||d||b> = R (-1)^{ja+1/2} \sqrt{[ja][jb]} \, tjs(ja,jb,1, -1/2,1/2,0)\]

\[<a||d||b> = R <k_a||C^k||k_b>\]

\[R = -e \int r(f_a f_b + g_a g_b) \, dr\]
