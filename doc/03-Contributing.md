\page contributing Contributing

\brief How to contribute to ampsci.

Updates, additions, changes, bug reports, and feature requests are all welcome.
Please follow the guidelines below to help keep the codebase clean, consistent, and maintainable, and to make merging your changes easier.

# Bugs and Feature Requests

\brief How to report bugs and request features via GitHub Issues.

* Submit bugs, feature requests, and questions via [GitHub Issues](https://github.com/benroberts999/ampsci/issues).
* For bugs: include a minimal input file that reproduces the problem, the ampsci version (or commit hash), and the error/unexpected output.
* For feature requests: a short description of what you need and why is sufficient.
* If you notice any compiler warnings when compiling ampsci, please submit those as bugs.
